# Mood-Spend
A C++ console app that tracks expenses by mood. Log your spending, tag it with how you're feeling, and spot emotional patterns in your habits. Built with all four OOP pillars — encapsulation, abstraction, inheritance, and polymorphism.
## Features
- Track expenses
- Tag with mood
- View emotional spending patterns

## How to Run
1. Compile:
   g++ mood_spend.cpp -o mood
2. Run:
   ./mood

## Concepts Used
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
